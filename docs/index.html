RocIBCFD

An OpenFOAM-based software tool for Internal Ballistic simulations of solid rocket motors.

Advanced Multiphase Flow Simulation for Rocket Propulsion

RocIBCFD is a comprehensive, OpenFOAM-based framework for simulating high-speed, gas-particle multiphase flows, specifically tailored for solid rocket motor (SRM) applications.

It's a powerful tool for researchers and engineers to study complex internal ballistics, model propellant regression, and accurately evaluate key performance metrics like thrust, C*, and Isp.

ðŸš€ Core Simulation Capabilities

The framework's core is a robust Euler-Euler multiphase flow solver that solves separate mass, momentum, and energy equations for both the gas and particle phases. This approach allows for the high-fidelity resolution of complex physics inside the rocket motor, including:

  • High Speed Flows: Capable of simulating two-phase (gas-particle) and single-phase flows through convergent-divergent (CD) nozzles (for flows with Mach number < 3.5).
  • Propellant Regression: Accurately models the regressing propellant surface for common grain geometries such as center-perforated and end-burning configurations (future releases will handle arbitrary grain geometries).
  • Interphase Interactions: Momentum and energy exchange (drag and heat transfer) between the gas and particle phases for all flow regimes that includes inertial, compressible and rarefield flow regimes.
  • Interparticle Interactions: Models collisions and interactions between particles.

To accelerate your research, this suite is more than just a solver. It also includes:

  • Specialized Boundary Conditions: A library of boundary conditions specifically developed for high-speed propulsion simulations.
  • Performance Utilities: Handy tools to help set up initial conditions and post-process results to extract key performance parameters.
  • Tutorial and Validation Cases: A set of tutorial and validation cases â†— to demonstrate the software's capabilities and help new users get started quickly.

Quick Installation

Get started by cloning the repository and running the compile script. Requires a working OpenFOAM installation (e.g., v2112).

  1. 1. Ensure OpenFOAM-v2112 is sourced

    Ensure your OpenFOAM installation (e.g., v2112) is installed and sourced in your terminal. If you need to install it, you can follow the official OpenFOAM-v2112 build instructions â†—

  2. 2. Clone the repository and enter the directory 

    $ git clone https://github.com/Ganeshkumar-V/Propellant-Regression-Framework.git
    $ cd Propellant-Regression-Framework

  3. 3. Source the project 'bashrc' 

    $ source bashrc

  4. 4. Compile the project 

    $ ./Allwmake

    (Optional: run ./Allwclean first to clean the project.)
